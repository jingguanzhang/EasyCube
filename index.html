EasyCube 易魔方

EasyCube 易魔方是一款免费且易用的多阶魔方还原工具，支持 2~50 超多阶数求解。内置国际主流算法，可快速解算还原步骤，并通过 3D 动画实时演示每一步操作，让复杂魔方也能轻松还原。应用支持自由打乱、自定义涂色、步骤播放、进度保存、PDF 教程导出等丰富功能，无论是魔方新手还是进阶玩家，都能快速掌握还原技巧。更支持相机扫描与图片识别等功能，助你随时随地练习与挑战魔方。

快速上手
自由模式
  1. 选择魔方阶数（2×2 ~ 50×50）
  2. 点击 打乱 生成随机状态
  3. 点击 求解 启动算法
  4. 使用底部导航栏 播放 / 逐步 查看还原
  5. 左侧步骤列表可 点击跳转 到任意步骤
自定义模式
  1. 切换到 自定义模式
  2. 在右侧 2D 展开图上 手动涂色
  3. 或点击面旁 用 相机 / 图片 自动识别
  4. 每种颜色填满 N×N 块后自动解锁求解
